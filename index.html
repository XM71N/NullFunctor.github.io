How to start with BitcoinVG?

BitcoinVG is developed to provide quick payments and store of value via the one minute Proof of Transaction (PoT) in every block. It is always synched to the latest Bitcoin release while adding the one minute PoT feature ensuring 100% Bitcoin code quality. BitcoinVG improves upon Bitcoin's slow block time making BVG suitable for quick payments.
